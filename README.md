# TextLoom

**TextLoom** is a minimal-setup, frictionless WebUI for generating high-quality synthetic data using local or API-based Large Language Models (LLMs). It solves the complexity of mock data creation by providing an intuitive, ready-to-use interface—no complex scripts required.

## Key Features

*   **Minimal Setup:** Get started immediately with zero configuration headaches.
*   **Local & API Support:** Run models locally for privacy or connect to powerful external APIs.
*   **Synthetic Data Generation:** Easily design and instruct models to produce structured, high-quality mock data.
*   **Modern, Smooth UI:** Enjoy a premium, Anthropic Claude-inspired aesthetic with buttery-smooth interactions.
